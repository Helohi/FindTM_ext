# Расширение FindTM
FindTM_ext - это расширение для браузера Chrome с открытым исходным кодом, которое позволяет проверять доступность ссылки, не открывая ее в браузере.
Это расширение помогает пользователям из РАЗных стран экономить время
 
## Руководство по установке
Чтобы установить это расширение в свой браузер Chrome, вам необходимо:
1. Перейти на страницу [Релизы](https://github.com/Helohi/FindTM_ext/releases) и загрузить последнюю версию *FindTM_ext.zip*
2. Извлечь файлы из *FindTM_ext.zip*
3. Открыть браузер Chrome
4. Перейдите в [chrome://extensions/](chrome://extensions/)
5. В правом верхнем углу выберите "*Режим разработчика*" (*Developer mode*).

<img width="145" height="33" alt="image" src="https://github.com/user-attachments/assets/ad213cea-248a-4dbf-ad74-397b5e88c008" />

6. В открывшейся верхней панели выберите "*Загрузить распакованное*" (*Load unpacked*).

<img width="133" height="49" alt="image" src="https://github.com/user-attachments/assets/e6283a3e-974e-45ce-b59d-262685f3b495" />

7. Выберите извлеченную папку из пункта [2] (Папка должна содержать "*manifest.json*") (**FindTM_ext**)
 
8. Все готово! Перейдите к [google.com](https://google.com/search?q=FindTM_ext+Helohi+github) и поищите что-нибудь!
